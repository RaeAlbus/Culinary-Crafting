# Culinary Crafting

A single-player puzzle game where players combine ingredients to craft delicious dishes. Inspired by *Little Alchemy*, **Culinary Crafting** challenges players to experiment with ingredient combinations in a guided cooking experience.

Each level presents players with a target dish and a set of base ingredients. Through trial and error, players must discover the correct combinations and cooking steps to create the dish. The game provides hints along the way, but the discovery process is up to you!

**Gameplay Features:**
- Drag-and-drop ingredient combining system
- Progressive difficulty across multiple levels

- Audio cues for crafting success and failure
- Intuitive slot-based crafting interface
